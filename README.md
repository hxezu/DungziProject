## 👨‍👩‍👧‍👦 About Dungji

둥지는 전 세대가 사용하는 SNS의 기능을 모은, 프라이빗한 우리 가족만의 SNS입니다.<br>
벌어진 가족 간의 소통을 다시 살려보고자 시공간의 제약을 뛰어넘어 디지털 기술로 이를 연결하고자합니다.
